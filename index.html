üìå Platform Overview

The Bitcoin Time Capsule facilitates the creation of on-chain, time-locked messages within the Bitcoin Signet test network. Messages are Base64 encoded and remain inaccessible until a predetermined future block height. Upon unlock, a portion of accumulated fees is distributable to early participants retrieving these messages.

üö® Important Notice: Signet Test Network

This application operates exclusively on the Bitcoin Signet test network. All transactions utilize Signet BTC (tBTC), which possesses no real-world monetary value. Complimentary Signet BTC can be obtained from a recognized Signet Faucet.

üîπ Wallet Configuration for Signet

To interact with this platform, your Bitcoin wallet must be configured for the Signet network:
